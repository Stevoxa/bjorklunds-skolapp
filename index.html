JSON-format

Appen läser dessa filer som JSON. Vid import måste strukturen matcha exemplen nedan.

1) Barnets schema (t.ex. schema_mikaela.json)

{ 
  "elev": "Elev A (exempel)",
  "schema": [
    {
      "dag": "Måndag",
      "lektioner": [
        { "tid": "09.00-10.00", "ämne": "Matematik (Sal A1)", "lärare": "Lärare Exempel 1" }
      ]
    }
  ]
}

2) Lov & helgdagar (t.ex. lov_helg.json)

{
  "lov": [
    { "datum": "2026-02-23", "vecka": 9, "dag": "Måndag", "beskrivning": "Exempel-lov" }
  ],
  "helgdagar": [
    { "datum": "2026-04-03", "vecka": 14, "dag": "Fredag", "beskrivning": "Exempelhelgdag" }
  ]
}

3) Terminer (t.ex. termin.json)

[
  {
    "termin": "Vårterminen 2026 (exempel)",
    "skolstart": [{ "datum": "2026-01-12", "beskrivning": "Exempel: vårterminens första skoldag" }],
    "skolavslutning": [{ "datum": "2026-06-11", "beskrivning": "Exempel: vårterminens sista skoldag" }]
  }
]

4) Paket (t.ex. skolapp-bundle.json)

Exemplet nedan är annoterat (med kommentarer). Importen kräver giltig JSON utan kommentarerna (dvs ta bort `// ...` rader innan import). Använd gärna knappen nedan för att ladda ner en importbar mall. 
{
  "bundleVersion": 1,                // paketversion (metadata)
  "appVersion": "v51",             // metadata (matchar din appversion)

  "children": [                    // barn (i appen = samma som "Elev A/B")
    {
      "id": "elev_a",             // valfritt men rekommenderas (stabil identifierare)
      "name": "Elev A",          // visas i UI
      "rss": "https://...",     // RSS-länk för matsedel (används av matsedel-funktionen)

      "schema": {
        "schema": [
          {
            "dag": "Måndag",      // måste matcha svensk veckodag (t.ex. "Måndag", "Tisdag"...)
            "lektioner": [
              {
                "tid": "09.00-10.00",     // format: "HH.MM-HH.MM" eller "HH:MM-HH:MM"
                "ämne": "Matematik (Sal A1)", // text som visas; "Sal" plockas ur parentes
                "lärare": "Lärare Exempel 1"   // valfritt, kan utelämnas
              }
            ]
          }
        ]
      },
      "contacts": [
        {
          "name": "",
          "email": "",
          "mobile": ""
        }
      ]
    }
  ],

  "lov": {
    "lov": [
      {
        "datum": "2026-02-23",  // format: YYYY-MM-DD (matchas mot visat datum)
        "vecka": 9,              // metadata (appens logik använder främst "datum")
        "dag": "Måndag",         // metadata
        "beskrivning": "Lov-text" // visas i appen
      }
    ],
    "helgdagar": [
      {
        "datum": "2026-04-03",
        "vecka": 14,
        "dag": "Fredag",
        "beskrivning": "Helgdag-text"
      }
    ]
  },

  "terms": [
    {
      "termin": "Vårterminen 2026", // metadata
      "skolstart": [
        { "datum": "2026-01-12", "beskrivning": "Skolstart-text" }
      ],
      "skolavslutning": [
        { "datum": "2026-06-11", "beskrivning": "Skolavslutning-text" }
      ]
    }
  ]
}

5) Kontaktlista (t.ex. larare_kontakt.json)

Förväntad struktur är en JSON-array med objekt. Fält: name (namn), email (valfritt men rekommenderas), mobile (valfritt). 
[
  {
    "name": "",
    "email": "",
    "mobile": ""
  },
  {
    "name": "",
    "email": "",
    "mobile": ""
  }
]

Installera som app (iOS)

iOS visar ofta ingen automatisk “installera”-prompt från PWA-kod. Gör så här istället:
1. Öppna appen i Safari på iPhone.
2. Tryck på Dela-ikonen (fyrkant med pil).
3. Välj Lägg till på hemskärmen.
4. Bekräfta. Ikonen dyker upp på startskärmen.

Installera som app (Android)

På Android/Chrome får du ofta en install-prompt när du är online. Om den inte dyker upp direkt kan du trycka på webbläsarens meny och välja “Installera app” eller ladda om sidan och vänta en stund.
